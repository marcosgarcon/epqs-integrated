Guia de Utilização - Gerenciador de Projetos APQP

Esta ferramenta foi desenvolvida para ser um gerenciador de projetos completo, focado nas 5 fases do Planejamento Avançado da Qualidade do Produto (APQP).

Tela Principal (Dashboard)

Esta é a primeira tela que você vê. Ela resume todos os projetos APQP que você salvou.

  • Para iniciar um novo projeto, clique no botão + Novo Projeto.
  • A barra de progresso de cada projeto na lista oferece uma visão rápida do seu andamento geral.
  • Você pode Editar para abrir e continuar trabalhando em um projeto ou Excluir para removê-lo permanentemente.

Gerenciando um Projeto

Ao criar ou editar um projeto, você entra na tela de gerenciamento detalhado.

Cronograma Visual (Timeline)

No topo, um cronograma visual (similar a um gráfico de Gantt) mostra a duração planejada de cada uma das 5 fases. Este gráfico é gerado automaticamente com base nas datas de início e fim que você define para as tarefas dentro de cada fase.

Fases do APQP (Acordeão)

O corpo principal do projeto é dividido nas 5 fases. Clique no título de uma fase para expandir ou recolher seu conteúdo.

  • Dentro de cada fase, há uma tabela para as tarefas (ou entregáveis) daquela etapa.
  • Use o botão + Adicionar Tarefa para incluir novos itens que sejam específicos do seu projeto.

Trabalhando com Tarefas

Cada tarefa na tabela possui campos detalhados para um gerenciamento eficaz:

  • Responsável: Defina quem é o dono da tarefa.
  • Data Início e Fim: Essencial para o cálculo do cronograma visual.
  • Status: O coração do acompanhamento. O status Concluído atualiza as barras de progresso. Se a data fim de uma tarefa passar e ela não estiver como 'Concluído', seu status será automaticamente mudado para Atrasado.
  • Anexos: Clique no botão Anexos para abrir uma área de upload. Você pode anexar múltiplos arquivos (relatórios, FMEAs, planilhas, etc.) como evidência para cada tarefa.

Funcionalidades Gerais

  • Salvar Projeto: Suas alterações são salvas ao clicar neste botão.
  • Backup e Restauração: Use os botões no cabeçalho para baixar um arquivo .json com todos os seus projetos ou restaurar a partir de um backup. Faça backups regularmente!
  • Exportar PDF: Com um projeto aberto, clique em "Exportar PDF" para gerar uma versão para impressão ou para salvar como PDF.

Aviso Importante: Todos os dados são salvos no seu navegador. Limpar o cache ou os dados de site irá apagar seus projetos. Utilize a função de backup com frequência.
