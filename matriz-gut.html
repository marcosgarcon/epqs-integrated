Guia de Utilização - Matriz de Priorização GUT

Esta ferramenta transforma a Matriz GUT de uma simples tabela de classificação em um sistema completo de priorização e ação estratégica.

Fluxo de Trabalho

A ferramenta é dividida em 3 etapas claras, acessíveis pelas abas no topo do formulário:

  1. Matriz GUT: Liste e pontue os problemas.
  2. Gráfico de Prioridades: Visualize os resultados para identificar o que é mais crítico.
  3. Plano de Ação: Crie tarefas para resolver os problemas priorizados.

1. Aba "Matriz GUT"

  • Use o botão + Adicionar Problema para criar novas linhas na tabela.
  • Problema: Descreva de forma clara e concisa cada item a ser analisado.
  • G, U, T: Para cada problema, selecione uma nota de 1 a 5 para Gravidade, Urgência e Tendência. Use a seguinte referência:
    • 5: Extremamente Grave / Urgente / Piora Rapidamente
    • 4: Muito Grave / Urgente / Piora em Curto Prazo
    • 3: Grave / Urgente / Piora
    • 2: Pouco Grave / Urgente / Piora em Longo Prazo
    • 1: Sem Gravidade / Pode Esperar / Não Piora
  • O Score é calculado automaticamente (G x U x T).
  • Clique em Ordenar por Prioridade para reorganizar a tabela, colocando os problemas com maior score no topo.

2. Aba "Gráfico de Prioridades"

Esta aba mostra um gráfico de barras com os problemas já ordenados por prioridade. É uma forma visual e rápida de entender e apresentar quais itens exigem atenção imediata.

3. Aba "Plano de Ação"

Após identificar os problemas mais críticos, esta aba permite que você crie um plano de ação para resolvê-los. Use o botão + Adicionar Ação para criar tarefas, definir responsáveis, prazos e acompanhar o status de cada uma.

Funcionalidades Gerais

A ferramenta conta com um Dashboard para gerenciar todas as suas matrizes, e as funções de Backup, Restauração e Exportação para PDF no cabeçalho.
